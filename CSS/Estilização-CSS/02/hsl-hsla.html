Entendendo hsl() e hsla() no CSS

O modelo HSL descreve cores por Matiz (Hue), Saturação (Saturation) e Luminosidade (Lightness). A variante HSLA adiciona o Alpha (opacidade).

O que cada componente significa?

Hue (H) — o ângulo no círculo cromático (0–360). 0 = vermelho, 120 = verde, 240 = azul.

Saturation (S) — intensidade da cor (0% = cinza, 100% = cor viva).

Lightness (L) — luminosidade (0% = preto, 50% = cor normal, 100% = branco).

Alpha (A) — opacidade (0 = totalmente transparente, 1 = opaco). Apenas em hsla() ou hsl() com barra (hsl(200 80% 50% / .5)).

Exemplos prontos

hsl(0 100% 50%) // vermelho
hsl(120 100% 40%) // verde
hsl(240 100% 50%) // azul
hsl(200 80% 50% / .5) // HSLA (50% opacidade)
hsl(45 100% 50%) // amarelo
hsl(300 80% 70%) // magenta claro

Sintaxes aceitas no CSS moderno

Ver exemplos de sintaxe 
/* Sintaxe tradicional (com vírgulas) */
color: hsl(200, 80%, 50%);

/* Sintaxe moderna (space-separated) */
color: hsl(200 80% 50%);

/* Com alpha (HSLA) — duas formas equivalentes */
color: hsla(200, 80%, 50%, .4);
color: hsl(200 80% 50% / .4);

/* Variáveis CSS também funcionam */
--h: 200; --s: 80%; --l: 50%; --a: .6;
color: hsl(var(--h) var(--s) var(--l) / var(--a));
Dica: combine HSL com variáveis CSS para temas escuros/claros alterando apenas --l (luminosidade) e --s (saturação).
