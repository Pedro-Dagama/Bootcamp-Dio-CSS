Propriedades de Fontes

font-family
font-size
font-weight
font-style
font-variant
font-stretch
line-height
font (shorthand)

font-family: define o tipo de fonte.
font-size: define o tamanho da fonte.
font-weight: define o peso (espessura) da fonte.
font-style: define o estilo (normal, itálico, etc).
font-variant: define variantes como small-caps.
font-stretch: estica ou comprime a fonte.
line-height: define o espaçamento entre linhas.
font: permite definir várias propriedades em uma linha.
