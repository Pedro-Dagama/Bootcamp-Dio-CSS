text-align: alinhamento do texto.
text-transform: transforma o texto (maiúsculas, minúsculas, capitalizado).
text-decoration: decoração do texto (sublinhado, linha sobreposta, etc).
text-indent: recuo da primeira linha.
letter-spacing: espaçamento entre letras.
word-spacing : espaçamento entre palavras.
line-height: altura da linha.
direction: direção do texto (ltr ou rtl).
white-space: como o espaço em branco é tratado.
