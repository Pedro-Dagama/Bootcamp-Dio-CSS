Propriedades de Sombras

box-shadow
text-shadow
múltiplas sombras
filter: drop-shadow

box-shadow: adiciona sombra em caixas (div, imagens, etc).
text-shadow: adiciona sombra em textos.
múltiplas sombras: é possível combinar várias sombras separando por vírgula.
filter: drop-shadow: aplica sombra como filtro, ideal para imagens com transparência. A sombra respeita as áreas transparentes do elemento.
Sintaxe: filter: drop-shadow(offset-x offset-y blur-radius cor);
