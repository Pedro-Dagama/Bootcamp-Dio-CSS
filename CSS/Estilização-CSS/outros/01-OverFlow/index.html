Propriedade overflow

overflow: visible
overflow: hidden
overflow: scroll
overflow: auto

overflow controla o que acontece quando o conteúdo de um elemento ultrapassa seu tamanho:
visible: o conteúdo excedente fica visível.
hidden: o conteúdo excedente é ocultado.
scroll: adiciona barras de rolagem sempre.
auto: adiciona barras de rolagem apenas quando necessário.
