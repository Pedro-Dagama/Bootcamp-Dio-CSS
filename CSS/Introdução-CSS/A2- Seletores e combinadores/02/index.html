Agrupamento de Seletores

Podemos aplicar as mesmas regras CSS para vários Seleores diferentes. Para isso, podemos separar os seletores utilizando uma virgula(,)

Assim, não precisamos ficar copiando e colando os mesmos codigos para todos, nós podemos fazer um bloco só de CSS e reutilizar as mesmas regras!
