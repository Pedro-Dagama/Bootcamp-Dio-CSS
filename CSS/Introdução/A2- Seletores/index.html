Seletores CSS

Bem-Vindo!

No CSS nós possuiamos os seletores por ID's

  • Seletor por Tipo: Busca elementos por uma tag HTML
  • Seletor por ID(#): Busca elementos através do atributo "id"
  • Seletor por Classe(.):Busca elementos através do atributo "class"
  • Seletor Universal(*)Seleciona todos os elementos HTML
  • Seletor de atributo([atributo]) Seleciona elementos que possuem um atributo especifico em nosso documento HTML e conseguimos também buscar atributos com um valor especifico
